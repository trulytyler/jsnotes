The Basics
  • Make sure to add document ready to the top of the main.js file so that it renders the html page first and then the js

    $(document).ready(() => { });
  • Make sure to add CDN for jquery before the main.js file at the bottom of the html page, but before the end of the body tag

    <script src="https://code.jquery.com/jquery-3.2.1.min.js" integrity="sha256-hwg4gsxgFZhOsEEamdOYGBf13FyQuiTwlAQgxVSNgt4=" crossorigin="anonymous"></script>
  • In order to enter HTML that will not be parsed as HTML, you need to encode characters that are special to HTML. For instance:
    < is encoded as "&+lt"
    > is encoded as "&+gt"
    & is encoded as "&+amp"
  • the $ symbol signifies it's Jquery. Its best practice to name a jquery variable with $ in the beginning so people know what kinda code it is.
  • Variables are where people store objects like funtions!
  • When you're making a jquery function you use $() and you can target classes or ids so like a div id or a css class. So an example would be $('.classexample') or $('dividname').
  • You can take that one step further and add a method to it.There are sooo many methods to pick from so you can look them up. So an example would be $('#dividname').hide(); an even better example would be like this because it shows where you input code with document ready. 

    $(document).ready(() => {
  $('#nav-dropdown').hide();
});
